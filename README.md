Wunix is a developer terminal command pack designed to improve productivity (and maybe some fun commands too).
### Philosophy

Wunix focuses on developer utility commands combined with lightweight entertainment commands (mostly linux inspired).

The project is open source and welcomes community contributions.

Note: Wunix is not a terminal replacement.

---

## Contribution

- New commands are welcome.
- Prefer self-contained command scripts.
- Use the `Source/` folder when modifying executable implementations.
- Submit changes through GitHub pull requests.
